# Standing Instructions — Apply Before Every Task

Before starting ANY work in this project, follow these steps without exception:

1. Identify and list every file path that will need to be opened, read, or modified to complete the requested task.
2. Present that list to the user and ask for explicit permission before touching any files.
3. Wait for the user to confirm before proceeding.

**This applies to every task, every session, every time — no exceptions.**
Do not open, read, create, or modify any file until the user has reviewed and approved the list of files to be touched.
